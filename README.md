# Doodle Dash

A short game where the player has to navigate obstacles by placing their own platforms!

[Play in your browser here!](https://fire-entity.itch.io/doodle-dash)

## Controls:

  - AS/Left-Right | Move
  - W/Up/Space | Jump
  - S/Down/Ctrl/Cmd | Ground slam
  - Click | Place platform
  - R/T | Reset level
